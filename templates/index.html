Welcome to Bouncey Castle

This tiny service fetches the content from another URL and bounces it back to you.

Usage

Send a request to this service with a bounc_url query parameter:

https://<your-domain>/?bounc_url=https://http.cat/404

Add debug=1 to see request details and the fetched HTML in a toggleable pane.

Example

Try bouncing a funny website: / ?bounc_url=https://http.cat/418&debug=1
